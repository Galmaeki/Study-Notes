## JS 문법
#### 변수선언
- const : 재할당 금지
  - 참조고정이므로 객체 배열은 내용 변경 가능
- let : 재할당 가능
```js
const a = 1;
// a = 2; //  재할당 불가

let b = 1;
b = 2; //가능

const obj = [1,2,3]
obj.push(4) // 가능
```

#### 함수선언
```js
function f1() {}
const f2 = function (){};
const f3 => () => {};
```
- 변수선언문인 경우 세미콜론 필요
- 함수선언문인 경우 세미콜론 필요없음

#### 구조분해할당
- 배열 구조 분해
```js
const foods = ["김밥", "라면", "돈까스"];
const [a, b] = foods; // a="김밥", b="라면"
```
- 객체 구조 분해
```js
const item = { id: 3, name: "돈까스", price: 9000 };
const { name, price } = item; // name="돈까스", price=9000
```
#### 전개구문
```js
const a = [1, 2];
const b = [...a, 3]; // [1,2,3]
```
```js
const user = { name: "재현", level: 1 };
const next = { ...user, level: 2 }; // level만 바꾼 새 객체
```

#### null 처리
```js
const user = null;
console.log(user?.profile?.email); // 에러 안 나고 undefined
```
```js
const nickname = null;
const display = nickname ?? "익명"; // "익명"
```
