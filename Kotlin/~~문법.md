# Kotlin 문법

## 변수

#### 기본형
```kotlin
val a: Int = 1
```
#### 타입 생략 가능
```kotlin
val b = 1
```
#### 선언 & 초기화 분리 가능
```kotlin
val c: Int
c = 3 // 지연 할당시 타입 명시 필요 
```

### val
- value
- 값 초기화 이후 재할당 불가
- read-only
- 객체 자체의 불변성을 보장하지는 않음
- Java의 Final과 유사
```kotlin
val s = "변경 불가"
//s = "변경 가능?" // 컴파일 불가
```

### var
- variable
- 값 재할당 가능
- 타입 변경 불가
```kotlin
var s = "변경 가능"
s = "변경 가능 22"
```

### 탑 레벨 선언
- top-level declaration
```kotlin
var i = 1
fun main() {
    i+=1
    println(i)//2
}
```

## 함수
#### 기본형
```kotlin
fun sum(a: Int, b: Int): Int {
    return a + b
}
```
#### 표현식
```kotlin
fun sum(a: Int, b: Int): Int = a + b
fun sum2(a: Int, b: Int) = a+b// 타입 추론이 가능한경우 반환타입 생략 가능
```
#### Unit
- 반환타입이 없는 함수의 기본형
```kotlin
fun a (a:Int) {
    println(a)
}
```
- 의미 있는 반환값이 없는 함수의 반환 타입
- 반환 타입이 `Unit`이면 일반적으로 생략 가능
```kotlin
fun b (a: Int): Unit {
    println(a)
}
```

#### 디폴트 파라미터 
- Default Parameter
- 인자를 받지 않은 경우 디폴트 파라미터가 존재하는 경우 디폴트 값으로 작동함
```kotlin
fun main() {
    hi()//안녕!
    hi("HI")//HI
}

fun hi(message : String = "안녕!") {
    println(message)
}
```

## 반복문
### For
#### ForI
```kotlin
for (i in 0 .. 3){
    print(i)
}//0123
```
```kotlin
for (i in 0 until 3){
    print(i)
}//012
```
```kotlin
for (i in 0 .. 6 step 2){
    print(i)
}//0246
```
- step은 양수만 사용(음수인 경우 IllegalArgumentException 발생)
- 감소가 필요한 경우 downTo 사용
```kotlin
for (i in 3 downTo 0){
    print(i)
}//3210
```
```kotlin
for (i in 6 downTo 0 step 2){
    print(i)
}//6420
```

### 배열 & 컬렉션 순회
#### for-in
```kotlin
val nums = arrayOf(1,2,3)
for (i in nums){
    print(i)
}//123
```
#### forEach
```kotlin
val nums = arrayOf(1,2,3)
nums.forEach {n ->
    print(n)
}
// it 을 사용한 경우
nums.forEach {print(it)}
```

### While
```kotlin
var x = 3
while (x>=0){
    print(x--)
}//3210
```

## 흐름제어
### If
- Java와 비슷하게 사용
```kotlin
fun ifs(a: Int){
    if(a>5) println("5보다 큼")
    else println("5와 같거나 작음")
}
```
#### 표현식
- If 문법은 표현식임
```kotlin
val a = 5
val 판별: String =
    if (a > 5) {
        "a는 5보다 큼"
    } else if (a < 5) {
        "a는 5보다 작음"
    } else {
        "5와 같음"
    }
```
- 삼항 연산자가 존재하지 않음

### When
- Switch 문법과 비슷
```kotlin
val day = 1
val result = when (day) {
    1 -> "월요일"
    2 -> "화요일"
    3 -> "수요일"
    4 -> "목요일"
    5 -> "금요일"
    else -> "주말"
}
```
#### 모든 경우를 처리하면 else 생략 가능
```kotlin
fun main() {
    val day = true
    val result = when (day) {
        true -> 1
        false -> 2
    }

    val color = when (getColor()) {
        Color.RED -> 1
        Color.BLUE -> 2
    }

}

enum class Color {
    RED, BLUE

}

fun getColor() = Color.RED
```
#### 여러개의 조건을 한줄로 지정 가능
```kotlin
val day = 1
when (day) {
    1, 0 -> println("1 이나 0")
    else -> println("1 이나 0이 아님")
}
```
